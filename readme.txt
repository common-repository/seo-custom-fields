Plugin customized the title, add description and keywords for category and tag pages

== Description ==

SEO Custom Fields plugin allows you to add custom meta tags for category and tag pages. This plugin will be useful for you in case if your SEO plugin doesn't support setting of meta tags for category and tag pages.

You can override the title and set meta description and meta keywords for category and tag pages.

It doesn't demand spacial tuning or coding. Just usual installation

== Installation ==

= From your WordPress dashboard =
1. Visit 'Plugins > Add New'
2. Search for 'SEO Custom Fields'
3. Activate SEO Custom Fields from your Plugins page.

= From WordPress.org =
1. Download SEO Custom Fields.
2. Upload the 'SEO Custom Fields' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate SEO Custom Fields from your Plugins page.

= Once Activated =
1. Visit plugin options (Settings - SEO Custom Fields)
2. Set a limit number of symbols for meta tags, title and description.
3. Fill in appropriate tag words and description in additional fields which are appeared on categories and tag pages of WP menu after plugin installation.
4. That's it!
